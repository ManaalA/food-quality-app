## Features
- 📝 Student feedback form with live NLP sentiment analysis
- 🖼️ Food image upload with visual quality scoring
- 📊 Interactive dashboard with vendor trends & heatmaps
- 🚨 Complaint pattern detection
- 💡 AI-powered improvement suggestions for vendors

## How to Run Locally

```bash
pip install -r requirements.txt
python -m textblob.download_corpora
streamlit run app.py
```

## Deploy on Streamlit Cloud (Free)

1. Push this folder to a GitHub repository
2. Go to https://share.streamlit.io
3. Connect your GitHub repo
4. Set main file as `app.py`
5. Click Deploy — done! ✅
